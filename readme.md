Visualize :

- Audio Files (.mp3)
- Algorithms
- Graphs
- Wave functions

And more

Tech used : Opengl & Vulkan

### How to run

`audvis data.mp3`
`audvis data.csv`

### How build

 - Generate project files using premake
Run the premake script
`premake5 vs2017` for windows or `premake5 gmake2`

The solution/make files will be in the `workspace` directory.
